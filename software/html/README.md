# macro_numpad WebUI 使い方ガイド

## 概要

このドキュメントでは、`macro_numpad.html` を使ってキーボード本体のキーマップ・マクロを設定する手順を説明します。
`macro_numpad.js` がWebHID通信と画面制御を担当し、`style.css` がUIの見た目を担当します。

## 動作要件

- WebHID対応ブラウザ（Chrome / Edge 推奨）
- USB接続された macro_numpad デバイス
- この `html` ディレクトリ内ファイル一式

## 起動方法

1. `macro_numpad.html` をブラウザで開きます。
2. 画面上部の「キーボードに接続」を押します。
3. デバイス選択ダイアログで対象デバイスを選びます。
4. 接続に成功すると、キーマップとマクロが自動読み出しされます。

## 画面構成

- 接続ボタン: WebHIDでデバイス接続を開始します。
- キーマップ操作ボタン:
  - キーマップを読み出す
  - キーマップを書き込む
  - マクロを読み出す
  - マクロを書き込む
- マクロ編集ボタン: マクロエディタを開きます。
- テンキー表示領域: キーの現在割り当てと変更状態を表示します。
- 補助ボタン:
  - デフォルトにリセット
  - ブートローダー開始
  - NVMを読み出す
  - macro ptrs
- ログ領域: 操作結果、通信エラー、デバッグ情報を表示します。

## キーマップ設定の流れ

1. 変更したいキーをテンキー表示領域でクリックします。
2. キー選択ダイアログからキーを選びます。
3. 必要なら修飾キー（Shift/Ctrl/Alt/Win）を有効にします。
4. 設定後、キーに保留マーク（⏳）が表示されます。
5. 「キーマップを書き込む」を押してデバイスへ反映します。

### スイッチ0の設定

- スイッチ0は通常キーではなくモード制御用です。
- 特殊ダイアログで以下を選びます。
  - Num: テンキーモード
  - Fn: マクロパッドモード（レイヤー切替）

## マクロ設定の流れ

1. 「マクロを編集する」を押してマクロエディタを開きます。
2. 編集対象のスロット（m0〜m9）を選択します。
3. キー群からコードを順番に追加します。
4. 必要に応じて遅延コード（100ms/500ms/1sec）を追加します。
5. 間違えた場合は「最後を削除」または「クリア」で修正します。
6. 「マクロを書き込む」を押してデバイスに保存します。

### マクロ割り当て

1. 通常のキー設定ダイアログで割り当て対象キーを選びます。
2. マクロトリガー（m0〜m9）を選択します。
3. 「キーマップを書き込む」で保存します。

### 実行中断

マクロ送信中にいずれかのキーを押すと、送信中のマクロは停止します。

## NVM関連操作

- キーマップを読み出す: デバイスNVMのキー設定を読み込みます。
- マクロを読み出す: デバイスNVMのマクロデータを読み込みます。
- NVMを読み出す: 生データを16進表示します（デバッグ用途）。
- macro ptrs: マクロ先頭ポインタ情報を表示します（デバッグ用途）。
- デフォルトにリセット: キーマップ初期状態とマクロ消去を行います。

## 重要な注意点

- 接続中デバイスが抜線・リセットされた場合、操作は無効になります。再接続してください。
- キーマップ変更とマクロ変更は別保存です。必要に応じて両方書き込んでください。
- ブートローダー開始を実行すると、デバイスが一時的に切断されることがあります。

## トラブルシューティング

### デバイスが表示されない

- USBケーブルのデータ通信対応を確認してください。
- ブラウザのWebHID対応状況を確認してください。
- 他アプリが同じデバイスを占有していないか確認してください。

### 書き込みできない

- 先に接続が成功しているか確認してください。
- ログ欄のエラー（Timeout、Version mismatch など）を確認してください。
- 一度「キーマップを読み出す」後に再編集して再実行してください。

### マクロが期待どおり動かない

- マクロスロットの中身と割り当て先キーを再確認してください。
- 遅延コードの入れすぎで体感上遅くなっていないか確認してください。
- マクロ容量上限（45バイト）を超えていないか確認してください。
