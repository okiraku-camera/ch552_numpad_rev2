# ソフトウェア概要

## simple_numPad
CH552G向けの最小構成USB HIDテンキーファームウェアで、行列スキャンによる基本入力確認とハードウェア動作検証に適しています。

## macro_numpad
通常テンキーとして使えるだけでなく、WebHID対応WebUIからキー割り当て・2層キーマップ・最大10個のマクロを設定できる拡張版ファームウェアです。

## 共通事項
両者ともch55xDuino v0.25ベースのArduinoスケッチとして提供され、同一ハードウェアで用途に応じてシンプル運用と高機能運用を選べます。
詳細については各ディレクトリのREADMEを参照してください。
