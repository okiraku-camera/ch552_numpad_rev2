# Software Overview

## simple_numPad
simple_numPad is a minimal USB HID numeric keypad firmware for CH552G, suitable for basic matrix-scan input checks and hardware bring-up.

## macro_numpad
macro_numpad is an extended firmware that works as a normal keypad while allowing key assignment, a 2-layer keymap, and up to 10 macros via a WebHID-enabled Web UI.

## Common Notes
Both are provided as Arduino sketches based on ch55xDuino v0.25, so you can choose either simple operation or advanced macro operation on the same hardware.
For details, please refer to the README in each directory.
