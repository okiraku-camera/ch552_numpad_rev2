# macro_numpad（マクロパッド）

## 概要

macro_numpadは、CH552Gマイコンを搭載した**プログラマブルなテンキーパッド/マクロキーパッド**です。デフォルトではスタンダードなテンキーとして機能しますが、WebUIを使用してキーマップやマクロを自由に設定できます。

## 主な機能

- **テンキー/マクロパッド デュアルモード**：スイッチ0の設定によってテンキーまたはマクロパッドとして動作します。
- **2層レイヤーシステム**：スイッチ0（NumLockまたはFnキー）で2つの層を切り替え可能です。
- **マクロ機能**：10個のマクロ（m0～m9）を登録でき、複数のキーストロークを1つのキーに割り当てられます。
- **修飾キー対応**：Shift、Ctrl、Alt、Winキーとの組み合わせをサポートします。
- **WebUI設定**：WebHIDを使用してブラウザから直感的に設定できます。
- **永続ストレージ**：全設定はNVM（内蔵EEPROM : 128バイト）に保存されます。

## ハードウェア構成

### マイコン
- **CH552G**：低コスト8051互換マイコン。USB 2.0フルスピード対応。

### スイッチ配置
- **スイッチマトリックス**：4列×5行（物理的には20ポジション）
- **実装スイッチ数**：17個（スイッチID 0～19のうち3つは未使用）
- **スイッチ0**：特殊役割（NumLock/Fn トグル）
- **スイッチ1～16, 18, 19**：プログラマブルキー
- **未使用**：スイッチID 11, 17, 19

### LED
- **マルチファンクションLED**（ピン11経由）：
  - **テンキーモード時**：NumLock状態を表示（点灯 = NumLock ON）
  - **マクロパッドモード時**：現在のレイヤー状態を表示（点灯 = レイヤー1、消灯 = レイヤー0）

### 通信
- **USB HID（WebHID対応）**：PC側Webアプリケーションとの通信にWebHIDを使用

## 動作モード

### テンキーモード（デフォルト）
- スイッチ0が**NumLock（0x53）**に設定されている状態
- 両層に同じテンキーレイアウトを保持
- NumLock LEDが有効

### マクロパッドモード
- スイッチ0が**Fn（0xBC）**またはその他のキーに設定されている状態
- 層ごとに異なるキーマップを設定可能
- Fn キーで当該層を切り替え

## 使い方

### クイックスタート
1. ブラウザで `html/macro_numpad.html` を開きます。
2. 「キーボードに接続」を押してデバイスを選択します。
3. 必要なキーやマクロを設定し、「キーマップを書き込む」「マクロを書き込む」で保存します。

## ビルド方法

このファームウェアは以下の環境でビルドできます。

- Arduino IDE 1.8.19
- Visual Studio Code + vscode-arduino 拡張

いずれの方法でも CH55xDuino v0.25 が必要です。

### ボード設定

ボード設定メニューで以下を選択してください。

- USB Settings: `USER CODE w/ 148B USB ram`

この設定はHIDエンドポイントに必要なUSB RAMを確保するために必要です。

## 技術詳細

### NVM（128バイト）の使用方法
- **0～1バイト**：キーマップ有効フラグ（0xAA55）
- **2～41バイト**：レイヤー0のキーマップ（20キー × 2バイト）
- **42～81バイト**：レイヤー1のキーマップ（20キー × 2バイト）
- **82～126バイト**：マクロ文字列データ（ゼロ区切り）
- **127バイト**：マクロエリア終端（予約）

### キーコード形式
- **16ビット値**：下位8ビット = HID Usage ID、上位8ビット = 修飾キーフラグ
- **修飾フラグ**：
  - 0x01 = Left Ctrl
  - 0x02 = Left Shift
  - 0x04 = Left Alt
  - 0x08 = Left Win

### マクロ機能
- **最大10マクロ**（m0～m9）、最大45バイトのマクロストレージ
- **ゼロ区切り**：各マクロは\0で終端。マクロ間は単一の\0で区切られます
- **遅延コード**：0xDA（100ms）、0xDB（500ms）、0xDC（1秒）を挿入可能
- **マクロトリガー**：0xD0～0xD9（m0～m9）で10個のマクロをトリガー可能

## トラブルシューティング

### デバイスが接続できない
- ブラウザがWebHIDをサポートしているか確認（Chrome、Edge、Opera推奨）
- デバイスが認識されているか確認
- ブラウザの設定でUSBアクセス権限を許可

### キーマップが反映されない
- 「キーマップを書き込む」で変更を保存しているか確認
- 保留中の変更（⏳）がある場合は書き込み操作が必須

### マクロが実行されない
- マクロがm0～m9のいずれかに正しく割り当てられているか確認
- マクロストレージの残り容量を確認
- マクロシーケンスが正しいか確認（画面に表示されている内容を確認）
